Build a screen recording static site capable of independently recording multiple video streams and multiple audio streams at once

Video: 
Should be capable of recording
-From webcam
-From laptop screen
-From external monitor
-Individual tab (screen share should handle this)
-From a connected phone

Audio: 
Should be capable of recording
-From laptop microphone
-Laptop audio (SEPARATE TRACK)
-From external microphone


UI should be laid out as follows

Center: all video tracks laid out in a zoom meeting style grid/speaker setup
In grid: show basic resolution, time, is or is not recording data for all
buttons on corner of each for start, pause/resume, finish
when recording is paused, color the whole track red

In speaker: one main video in center, rest on bar on bottom
main video able to access more controls

Bottom bar: all audio tracks laid out in a single line

Left side: detailed audio/video config settings/inspector for one track (click track to select)
Right side: export settings


Export modes: 

single export: If only one video stream active, combine it with all audio streams. Disabled for multiple video streams.
export all: exports all videos and audios independently
merge export: assign manually one or multiple audio tracks to each video before exporting




Default configuration as follows: 

Video: main display

Audio: desktop audio and mic audio independently
